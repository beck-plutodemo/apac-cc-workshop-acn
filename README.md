## Student Setup

Before the workshop, complete these steps:

1. **Install Claude Code** — Follow the [official installation guide](https://docs.anthropic.com/en/docs/claude-code/overview)
2. **Fork the starter app** — Fork [beck-source/inventory-management](https://github.com/beck-source/inventory-management) to your own GitHub account
3. **Create an API key** — Get your Anthropic API key from [console.anthropic.com](https://console.anthropic.com)
4. **Clone your fork locally** and verify Claude Code runs in the project directory

Detailed instructions are in `guides/student-setup.md` (coming soon).
